# İçerik Doğallaştırıcı

voice-dna ile yazılan içeriğin son geçişi.
AI izlerini kaldır, Mehmet'in sesini koru.

## AI Yazı İzleri — Tespit Listesi

```
❌ "Bu bağlamda..."
❌ "Önemle belirtmek gerekir ki..."
❌ "Sonuç olarak..."
❌ Aynı uzunlukta birbirini izleyen cümleler
❌ Her paragrafta özet cümle
❌ "Kesinlikle", "mutlaka", "son derece"
❌ Pasif yapı: "yapılmıştır", "edilmiştir"
❌ Aşırı düzenli yapı: Giriş → 3 madde → Sonuç
```

## Doğallaştırma Teknikleri

```
Cümle ritmi değiştir:
  Uzun. Sonra kısa. Sonra biraz daha uzun bir cümle gelir.

Başlangıç çeşitlendir:
  "Bu...", "Bu...", "Bu..." → Farklı başlangıçlar kullan

Eksik bırak (kasıtlı):
  Her şeyi açıklama. Okuyucu tamamlasın.

Somutlaştır:
  "iyi performans" → "30+ FPS, mobilde bile"

Kişiselleştir:
  "geliştiriciler" → "sen" (doğrudan hitap)
```

## Mehmet'in Ses Koruyucuları

```
✓ Teknik terimler Türkçe kalır
✓ Kısa cümle ritmi bozulmaz
✓ Tarihi referanslar doğal gelir
✓ Abartı yok, alçakgönüllülük var
✓ "Denedim, şunu gördüm" formatı
```

## Örnek Dönüşüm

```
Önce (AI sesi):
"Bu oyun, Timurlu döneminin zengin kültürel mirasından
ilham alınarak geliştirilmiş olup son derece özgün bir
deneyim sunmaktadır."

Sonra (Mehmet'in sesi):
"Timurlu döneminden uyarladım. Kaynaklara baktım,
eksik kural setini mantıkla tamamladım. Tek HTML,
tarayıcıda çalışıyor."
```

## İş Akışı

```
voice-dna ile yaz → humanizer ile geçir → yayınla
```

Humanizer tek başına kullanılmaz — her zaman voice-dna sonrası.
